# Psicofarmacologia - Cerebro 3D

Visualizador didatico em Streamlit para mecanismos neurais da ansiedade e depressao.

## Rodar localmente

```powershell
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

## Estrutura

- `app.py`: aplicacao Streamlit standalone.
- `requirements.txt`: dependencias minimas.
- `.streamlit/config.toml`: configuracao visual basica do Streamlit.
